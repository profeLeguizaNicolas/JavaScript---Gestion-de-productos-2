JavaScript - Sistema de Gestión de Productos.

Se debe desarrollar un programa en JavaScript que permita administrar un listado de productos mediante un menú de opciones. Además, la opción 4 deberá desplegar un submenú con funcionalidades adicionales.



Datos iniciales

El programa parte del siguiente array de productos: 

1    const productos = [
2       {nombre: "Mouse", categoria: "tecnologia", precio: 1000, stock: 10 },
3       {nombre: "Teclado", categoria: "tecnologia", precio: 2000, stock: 5 },
4       {nombre: "Monitor", categoria: "tecnologia", precio: 50000, stock: 0 },
5       {nombre: "Silla", categoria: "hogar", precio: 15000, stock: 2 }
6   ];

Cada producto es un objeto con las propiedades.

  • nombre
  • categoria
  • precio
  • stock


Funcionamiento General.

El programa debe ejecutarse dentro de un ciclo (do-while) que muestre un menú de opciones.

Menú
  1. Agregar producto.
  2. Eliminar producto.
  3. Modificar producto.
  4. Más opciones.
  5. Mostrar productos y salir.

Submenú de funciones adicionales.

El sistema deberá mostrar un submenú con funcionalidades extra para trabajar el array de productos utilizando principalmente forEach()

En Más opciones mostrará las siguientes opciones:

SUBMenú
  1. Mostrar productos sin stock.
  2. Buscar productos por categoría.
  3. Aplicar aumento de precio por nombre.
  4. Mostrar producto más caro.
  5. Volver al menú anterior.

Detalle de cada funcion

  1. Mostrar productos sin stock

    El sistema deberá recorrer el array de productos y mostrar únicamente aquellos productos cuyo stock sea igual a 0.

    Los productos encontrados deberán mostrarse en formato de tabla en la consola.

    Además, esta opción deberá finalizar la ejecución completa del programa, saliendo tanto del submenú como del menú principal.

    Use forEach()


  2. Buscar productos por categoría

    Solicitar al usuario una categoría y mostrar todos los productos que pertenezcan a ella, por consola en formato tabla

    Esta opción deberá finalizar la ejecución completa del programa, saliendo tanto del submenú como del menú principal.

    Use forEach()


  3. Aplicar aumento por nombre.

    Solicitar al usuario el nombre de un producto y un porcentaje de aumento.

    Luego recorrer el array y generar un nuevo array de productos con los valores actualizados, manteniendo el principio de inmutabilidad y programación funcional, evitando modificar el array original.

    El nuevo array deberá conservar todos los productos originales y únicamente actualizar el precio del producto coincidente.

    Además, la opción Mostrar productos y salir deberá mostrar este nuevo array actualizado

    Use forEach()

  4. Mostrar producto más caro

    El sistema deberá recorrer el array de productos y mostrar el producto con el precio más alto utilizando console.table()

    Si previamente se aplicó la opción Aplicar aumento por nombre, el cálculo deberá realizarse sobre el nuevo array actualizado.

    En vaso contrario, deberá utilizarse el array original de productos.

    Use forEach()

  5. Volver al menú anterior

    El sistema deberá regresar al menú principal sin finalizar la ejecución del programa.
